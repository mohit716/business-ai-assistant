# Business AI Assistant

An AI-powered assistant designed to help business teams with day-to-day tasks — drafting emails, summarizing information, planning, and answering questions — built with Claude API and Streamlit.

## Live Demo
[Click here to try it](https://business-ai-assistant-uytmeb8v6lwwzwpnvjmemr.streamlit.app/)

## Features
- Conversational chat interface for non-technical business users
- Powered by Anthropic's Claude API
- Supports tasks across marketing, finance, operations, and service teams
- Deployable as a standalone web app or embeddable in internal tools

## Use Cases
- Draft follow-up emails and client communications
- Summarize reports and documents
- Answer business process questions
- Plan tasks and organize workflows

## Tech Stack
- Python
- Streamlit
- Anthropic Claude API
- Deployed on Railway

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deployed on Railway with environment variable `ANTHROPIC_API_KEY`
